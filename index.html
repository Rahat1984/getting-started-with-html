Important Concepts

How the Web Works

The web was invented in the early 90s. It is a collection of computers interconnected over a WAN that can communicate with each other. It contains roughly 30 billion documents.

HTML

HTML stands for Hypertext Markup Language. It is made up of:
Text content - this is what you see
Markup - This is what is looks like
Tags - This tells the markup what to do, and helps to form elements

Why Computers are Stupid

Programmers need to write code exactly how the computer understands otherwise the computer may well crash and/or fail to understand what has been inout. The programmer needs to write the correct syntax

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them, whereas inline elements do not begin on a new line when rendered visually.
